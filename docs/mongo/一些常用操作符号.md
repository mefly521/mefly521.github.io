常用运费符

| SQL             | MQL                                |
| --------------- | ---------------------------------- |
| a=1             | {a: 1}                             |
| a<>1            | {a: {$ne: 1}}                      |
| a>1             | {a: {$gt: 1}}                      |
| a<1             | {a: {$lt: 1}}                      |
| a>=1            | {a: {$gte: 1}}                     |
| a<=1            | {a: {$lte: 1}}                     |
| a=1 and b=1     | {a:1,b:1} 或 {$and: [{a:1},{b:1}]} |
| a=1 or b=1      | {$or:[{a:1},{b:1}]}                |
| a=null          | {a: {$exists:false}}               |
| a in (1,2,3)    | {a: {$in:[1,2,3]}}                 |
| a not in(1,2,3) | {a: {$nin:[1,2,3]}}                |
